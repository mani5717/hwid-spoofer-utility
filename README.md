Add the executable and its folder to your antivirus exclusions. You can verify the file hash against the published checksums in each release.

**Q: Does Hwid Spoofer work with Vanguard / Valorant?**

**Q: I use a laptop — will this damage my hardware?**
A: No. Hardware ID spoofing modifies registry entries and driver-level identifiers only — it does not write to BIOS/EFI or alter physical components. Full rollback is supported.
